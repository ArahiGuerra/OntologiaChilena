CONSULTAS PARA GRAFO (CONSTRUCT)

1. CANCIONES - ARTISTA - ÉPOCA

PREFIX : <http://www.semanticweb.org/sariw/ontologia_chilena#>
PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>

CONSTRUCT {
  ?cancion :tieneArtista ?artista ;
           :perteneceAEpoca ?epoca .
}
WHERE {
  ?cancion rdf:type :Cancion ;
           :tieneArtista ?artista ;
           :perteneceAEpoca ?epoca .
}
